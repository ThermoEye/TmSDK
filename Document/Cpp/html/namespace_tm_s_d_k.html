selfA pointer to the RoiSpot instance from which the spot is to be retrieved. This parameter must be a valid pointer to an existing RoiSpot object.
retA pointer to a boolean value that indicates whether the operation was successful. If the function completes successfully, this value is set to true. If an error occurs, it is set to false.
argA pointer to a Point structure where the coordinates of the spot will be stored. The function fills this structure with the spot's x and y coordinates.
Returns
A const char pointer to the error message if an exception is raised, otherwise nullptr.
