std::vector< TmRemoteCamInfo > TmSDK::TmRemoteCamera::GetCameraList ( )

Gets remote camera list.

Returns
Camera information list if this method gets connectable camera information successfully; otherwise, false if an exception is raised.

If you want to get all cameras that can be connected via multiple local network adapters

std::vector<TmRemoteCamInfo> list = RemoteCamera->GetCameraList();
