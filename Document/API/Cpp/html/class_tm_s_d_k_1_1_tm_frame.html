double TmSDK::TmFrame::GetPixel ( int  x,

)

Retrieves a single pixel value at the specified (x, y) coordinates from the raw frame.

Parameters
xThe x-coordinate (column index) of the pixel.
yThe y-coordinate (row index) of the pixel.
Returns

Returns the pixel value as a double. If the coordinates are out of bounds or the frame is null, returns 0.

Example usage:

double value = pFrame->GetPixel(10, 20);
printf("Pixel value: %f\n", value);

std::vector< std::vector< double > > TmSDK::TmFrame::GetPixel ( int  x,
int  y,
int  width,
int  height 
)

Retrieves a rectangular region of pixel data from the raw frame and returns it as a 2D array of doubles.

Parameters
xThe starting x-coordinate (left) of the region.
yThe starting y-coordinate (top) of the region.
widthThe width of the region to extract.
heightThe height of the region to extract.
Returns

Returns a 2D vector of doubles containing the pixel values within the specified region. If the frame is null, empty, or the region is out of bounds, an empty vector is returned.

Example usage:

std::vector<std::vector<double>> frame = pFrame->GetPixel(0, 0, width, height);
for (int h = 0; h < height; ++h) {
for (int w = 0; w < width; ++w) {
printf("%f ", frame[w][h]);
}
printf("\n");
}
