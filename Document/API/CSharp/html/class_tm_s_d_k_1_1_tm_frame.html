Gets a 2D array of raw values for the specified region.

Parameters
xThe starting x-coordinate of the region
yThe starting y-coordinate of the region
widthThe width of the region to retrieve
heightThe height of the region to retrieve
Returns
A 2D array of raw values with size [width][height]. Returns null if the operation fails.
// Get a 10x10 region of pixel values starting at (10, 10)
double[][] region = frame.GetPixel(10, 10, 10, 10);
if (region != null)
{
// Print the region values
for (int i = 0; i < region.Length; i++)
{
for (int j = 0; j < region[i].Length; j++)
{
Console.Write($"{region[i][j]}\t");
}
Console.WriteLine();
}
}
