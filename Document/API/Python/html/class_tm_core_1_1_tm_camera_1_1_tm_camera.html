◆ get_local_camera_list()

TmCore.TmCamera.TmCamera.get_local_camera_list ( )
static

Enumeration of local cameras.

Returns
A tuple containing:
  • names: A list of strings, where each string is the name of a local camera.
  • ports: A list of strings, where each string is the COM port of a local camera.
  • indexes: A list of integers, where each integer is the index of a local camera.
  • count: The number of cameras retrieved.
  • media_info_lists: A list of MediaInfo lists, where each list contains media information for a local camera.

◆ get_remote_camera_list()

TmCore.TmCamera.TmCamera.get_remote_camera_list ( )
static

Enumeration of remote cameras.

Returns
A list of RemoteCameraList objects, where each object contains:
  • name: A string, where each string is the name of a remote camera.
  • serial_number: A string, where each string is the serial number of a remote camera.
  • mac: A string, where each string is the MAC address of a remote camera.
  • ip: A string, where each string is the IP address of a remote camera.
  • adapter_ips: A list of strings, where each string is the adapter IP address of a remote camera.
  • count: The number of cameras retrieved.
  • media_info_lists: A list of MediaInfo lists, where each list contains media information for a remote camera.
